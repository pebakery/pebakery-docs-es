# DirMake

Creates a directory.

## Syntax

```pebakery
DirMake,<DestDir>
```

### Argumentos

| Argumento | Descripción |
| --- | --- |
| DestDir | Ruta completa del directorio para crear. |

## Observaciones

Ninguna.

## Relacionado

[DirCopy](./DirCopy.md), [DirDelete](./DirDelete.md), [DirMove](/DirMove.md)

## Ejemplos

### Ejemplo 1

```pebakery
// C:\Temp se creará.
DirMake,C:\Temp
```
